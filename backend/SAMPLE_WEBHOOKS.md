## 4. Grade Created Event
```json
{
  "event_id": "zoho_btec_grade_created_901234",
  "module": "BTEC_Grades",
  "event_type": "created",
  "record_id": "5843017000000901234",
```

## Notes:
- `event_id`: Unique ID generated by Zoho Workflow
- `record_id`: Zoho record ID (نستخدمه لـ fetch full data)
- Minimal payload - Backend fetches full record from Zoho API
